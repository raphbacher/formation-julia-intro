    Interoperability with other languages¶

    Python¶

    You can use Python packages from Julia with this Julia package : Pycall

    Example :

    julia> using PyCall
julia> @pyimport math
julia> math.sin(math.pi / 4) - sin(pi / 4)  # compares python math sin function with Julia's (hopefully returns 0.0)

    You can also Julia from Python with this package : https://github.com/JuliaPy/pyjulia

    The Julia docs provide also a list of noteworthy differences that have to be taken care of when coming from Python.

    R¶

    You can use R packages from Julia with this Julia package : RCall

    Example :

    julia> using RCall
julia> foo = 1
1

R> x <- $foo # by just pressing `$` we switch to R console !

R> x
[1] 1

    The Julia docs provide also a list of noteworthy differences that have to be taken care of when coming from Python.
